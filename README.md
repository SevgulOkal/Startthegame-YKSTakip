# 🚀 YKS Ders Takip Uygulaması

Bu proje, YKS hazırlık sürecini takip etmek için geliştirilmiş bir web uygulamasıdır.  
Kullanıcılar ders çalışma planlarını oluşturabilir, ilerlemelerini takip edebilir ve motivasyonlarını artırabilir.

---

## 🛠️ Kullanılan Teknolojiler

- React ⚛️
- Vite ⚡
- JavaScript (ES6+)
- React Router DOM

---

## 📦 Kurulum

Projeyi çalıştırmak için:

```bash
npm install


📌 Özellikler
Kullanıcı kayıt / giriş sistemi
Ders takip ekranı
Günlük çalışma planı
İlerleme takibi
Basit ve hızlı arayüz

🧠 Amaç

Bu proje, hem React öğrenmek hem de YKS çalışma sürecini daha düzenli hale getirmek için geliştirilmiştir.
